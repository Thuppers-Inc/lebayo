# Dossier pour l'application mobile

Placez votre fichier APK Android dans ce dossier avec le nom `lebayo.apk`.

Le fichier sera accessible via l'URL : `/app/lebayo.apk`

## Instructions

1. Renommez votre fichier APK en `lebayo.apk`
2. Placez-le dans ce dossier (`public/app/`)
3. La page de téléchargement sera automatiquement mise à jour
